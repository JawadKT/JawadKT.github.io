# JawadKT.github.io

Personal website for Jawad.

This repo is a static site designed to work directly on GitHub Pages (no build step).

## Structure

- `index.html` – main page with sections for Hero, About, Projects, Experience, Skills, and Contact
- `styles.css` – global styles, layout, and subtle animations
- `script.js` – small enhancements (active nav on scroll, mobile menu, copy-email button, dynamic year)
- `assets/` – placeholder for images and other static files

## Local preview

You can preview the site locally using Python's built-in HTTP server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000/` in your browser.
